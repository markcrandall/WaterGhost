# Water Ghost

A horror-themed multiplication game built as a single HTML file. It is well known that ghosts have to stop and count every correct answer you give them before they can keep haunting you...

## How to Play

- A ghost is approaching you across dark water
- Answer multiplication questions by clicking one of four stone tablets (or press 1-4)
- Correct answers push the ghost back
- Wrong or slow answers let the ghost creep closer
- If the ghost reaches you, it's over

## Ghost Stages

As the ghost gets closer, the atmosphere intensifies:

| Stage | Distance | What Changes |
|-------|----------|-------------|
| Far | 100%–60% | Faint silhouette, calm water ambient |
| Mid | 60%–20% | Face visible, wind and ghost voice kick in |
| Close | 20%–1% | Detailed face, heavy breathing, fog thickens |
| Scare | 0% | Jump scare — game over |

## The Catch

The ghost accelerates over time. Even a good player can only survive about 10 minutes. Correct answers buy time, but the ceiling drops — you can never push the ghost all the way back.

## Run It

Open `index.html` in any modern browser. No build step, no dependencies, no server required.

Works on desktop and mobile.
